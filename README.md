# BRCA1_ecDNA_custom_scripts
Custom scripts used in BRCA1_ecDNA project

trace_ampliconsuite_cnv_to_seeds.py: This script filters CNVkit results to generate AmpliconArchitect (AA) seeds and reports the specific filtering reason for each discarded CNV record. It provides the flexibility to bypass the standard arm-median-based threshold and BAM coverage-ratio check originally implemented in the AmpliconSuite-pipeline.

## Requirements:
AmpliconSuite-pipeline (https://github.com/AmpliconSuite/AmpliconSuite-pipeline) version 1.5.0

Python 3.10.18

## Installation:
No installation needed.

## Example Usage:
python3 trace_ampliconsuite_cnv_to_seeds.py --cns BZ2213695K_WT.cs.rmdup.cns --ref hg19 --aa_data_repo /path/to/AA_data_repo --gain 4.0 --cnsize_min 50000 -s BZ2213695K_WT --outdir ./BZ2213695K_WT 1> stdout.txt 2> stderr.log

The .cns file is generated by cnvkit, and it's generated in AmpliconSuite-pipeline by default.
